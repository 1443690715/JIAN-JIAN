# JIAN-JIAN
一款很棒的书摘软件 微信小程序 中山大学软件创新大赛十强参赛项目

## 产品介绍
笺，是小幅而华贵的纸张，

代表纸质书中精美的书摘，

笺笺， 是一款针对纸质书阅读的产品， 致力于简化记录书摘的过程， 随心所欲地采撷身边的美好。

这是一款重实用轻社交的产品， 也是一款重设计重技术的工业品， 这是一款注重情怀的应用，
